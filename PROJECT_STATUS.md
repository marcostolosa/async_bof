## 实现的功能

### 事件监控类型
1. **管理员登录监控** - 检测管理员权限获取
2. **进程启动监控** - 监控特定进程的启动
3. **用户登录监控** - 监控特定用户的登录
4. **网络隔离监控** - 检测网络连接断开
5. **VPN连接监控** - 监控VPN服务状态
6. **WiFi连接监控** - 检测WiFi连接状态
7. **票据过期监控** - Kerberos票据过期警告
8. **剪贴板监控** - 监控剪贴板内容变化
9. **键盘输入监控** - 简化的键盘活动检测
10. **用户登录事件监控** - 增强的用户登录检测

### 任务管理功能
- **任务创建** - 创建异步监控任务
- **任务列表** - 列出所有活动任务
- **任务取消** - 取消正在运行的任务
- **任务删除** - 从系统中完全删除任务
- **任务持久化** - 将任务保存到注册表
- **任务加载** - 从注册表恢复任务

### Cobalt Strike集成
- **右键菜单** - 在beacon上下文菜单中集成
- **图形界面** - 提供对话框进行任务配置
- **命令行别名** - 支持命令行操作
- **事件报告** - 将检测到的事件报告回团队服务器

## 使用方法

### Cobalt Strike使用
1. 加载 `async_bof.cna` 脚本
2. 右键点击beacon选择"Async BOF"菜单
3. 选择要添加的监控类型
4. 配置监控参数和检查间隔
5. 查看任务列表和事件报告

### 本地测试
```bash
cd d:\Github\async_bof
bin\async_bof_local.exe
```

### 命令行操作
```
async_bof                          # 加载现有任务
async_bof_add_task <name> <type> <params> <interval>
async_bof_list                     # 列出任务
async_bof_cancel <task_id>         # 取消任务
async_bof_delete <task_id>         # 删除任务
```
