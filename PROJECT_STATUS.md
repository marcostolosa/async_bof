# Async BOF - 项目状态

## 🎯 项目概述

**Async BOF** 是一个为Cobalt Strike设计的生产级异步Beacon Object Files (BOF)框架，提供真实的Windows系统事件监控能力。该项目已从概念阶段转换为完全可用的生产级工具。

### 监控模式
1. **轮询模式** (`MONITOR_MODE_POLLING`)
   - 传统的定期检查方式
   - 兼容性好，权限要求低
   - 适合基本监控需求

2. **事件日志模式** (`MONITOR_MODE_EVENT_LOG`)
   - 基于Windows事件日志
   - 实时响应，不遗漏事件
   - 需要适当的审计策略配置

3. **ETW模式** (`MONITOR_MODE_ETW`)
   - 内核级事件追踪
   - 最低延迟，最高性能
   - 需要管理员权限

4. **WMI模式** (`MONITOR_MODE_WMI`)
   - WMI事件订阅
   - 支持复杂查询条件
   - 良好的系统集成
